Moist Canvas Clean Package

How to use on Windows:
1. Extract the whole ZIP first. Do not run files inside the ZIP preview.
2. Double-click 安装依赖.bat.
3. Wait for it to download portable Python and install dependencies.
4. Double-click 运行文件.bat.
5. The browser will open http://127.0.0.1:6767/.
6. On first use, open the API settings page and enter your own API Key.

Notes:
- The first install requires internet access.
- You do not need to install Python manually.
- Close the run window to stop the local server.
- Generated images and videos are saved in the output folder.
- API keys stay on your own computer and are not included in this package.
