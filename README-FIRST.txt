Moist Canvas Test Package

How to use:
1. Extract the whole ZIP first. Do not run files inside the ZIP preview.
2. Double-click the dependency installer BAT file.
3. Wait for it to download portable Python and Python packages.
4. Double-click the run BAT file.
5. The browser will open http://127.0.0.1:3000/
6. On first use, set your API Key in the API settings page.

Notes:
- The first install requires internet access.
- Users do not need to install Python manually.
- Close the run window to stop the local server.
- Generated images and videos are saved in the output folder.
- API keys stay on the user's own computer and are not included in this test package.
